# BookHub — Backend

## Stack
Backend
- Java 17+
- Spring Boot 3.2+
- Spring Security + JWT
- Spring Data JPA
- SQL Server 2019+
- Gradle

## Configuration locale de la DB :
- Créer la BDD et l'appeler BOOKHUB
- ajouter un fichier `application-secret.properties` dans resources, le mettre dans le gitignore

```
spring.datasource.username=sa
spring.datasource.password=TonMotDePasseDocker!
```
